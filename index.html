1. Mengapa Menggunakan HTML, CSS, dan JavaScript?

1. HTML (HyperText Markup Language)

HTML adalah bahasa markup untuk membuat struktur halaman web. Elemen-elemen HTML seperti teks, gambar, dan tautan adalah kerangka utama halaman web.

2. CSS (Cascading Style Sheets)

CSS digunakan untuk mempercantik tampilan web, mengatur warna, font, tata letak, dan elemen visual lainnya agar halaman lebih menarik.

3. JavaScript

JavaScript menambahkan interaktivitas dan logika pada halaman web, seperti animasi, validasi formulir, dan memperbarui konten tanpa memuat ulang halaman.

2. Bagaimana HTML, CSS, dan JavaScript Diintegrasikan?

1. HTML: Struktur Halaman

HTML digunakan untuk mendefinisikan elemen-elemen seperti paragraf, gambar, tombol, dan lainnya. Semua komponen dasar halaman dibuat dengan HTML.

2. CSS: Gaya dan Tata Letak

CSS dihubungkan dengan HTML untuk memberikan gaya pada elemen-elemen seperti warna, margin, atau posisi. Hal ini dilakukan dengan file eksternal, internal, atau inline.

3. JavaScript: Interaktivitas

JavaScript digunakan untuk memberikan logika dan interaktivitas, seperti memperbarui teks atau menambahkan animasi ke halaman web.
